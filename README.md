# 码上生成 项目说明

## 网站简介
码上生成是一个由多个 AI 协作完成的完整网站，提供代码片段在线生成、预览与下载功能。页面采用响应式布局，兼容桌面与移动端，旨在帮助开发者快速获取常用语法模板，提升编码效率。

## 页面与功能
- 首页（index.html）：展示项目 LOGO、使用指南与快速入口。  
- 生成器页（generator.html）：选择语言、填写参数，实时预览生成的代码。  
- 模板库页（templates.html）：分类展示内置代码模板，支持搜索与收藏。  
- 关于页（about.html）：介绍团队成员、技术栈与项目背景。  
- 帮助页（help.html）：常见问题、使用技巧及反馈渠道。

## 多 AI 协作与验收
项目分为需求分析、UI 设计、前端实现与测试四个阶段，由不同 AI 负责对应模块。每完成一个阶段后，通过自动化单元测试与人工评审进行交叉验收，确保功能完整、界面一致且无明显错误。

## GitHub Pages 部署与访问（入口 index.html）
1. 将仓库推送至 GitHub 主分支（或 gh‑pages 分支）。  
2. 在仓库 Settings → Pages 中选择源分支并保存。  
3. GitHub 会自动构建并提供类似 `https://<username>.github.io/<repo>/` 的访问地址。  
4. 直接打开该地址即可访问 index.html，进入码上生成首页。
